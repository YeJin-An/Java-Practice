반복문
1. while문
-> 조건을 만족시키는 동안 블럭{ }을 반복
    - 반복횟수 모를 때,

while (조건식) {  수행될 문장 }
순서 : 조건식 -> 수행될 문장

2. do-while문
-> 블럭{ }을 최소한 한 번 이상 반복
    - 사용자 입력받을 때, 유용
do{ } while(조건식);
순서 : 수행될 문장 -> 조건식

// 사용자가 입력 받을 때, 주로 사용이 된다.
--> 입력한 결과를 다시 입력해야 하기 때문
// while문 보다 깔끔한 경우도 존재하기 때문이다.

break문
-> 자신이 포함된 하나의 반복문을 벗어난다.

contrinue문
-> 자신이 포함된 반복문의 끝으로 이동
    - 다음 반봇으로 넘어감
    - 전체 반복 중에서 특정 조건시 반복을
       건너뛸 때 유용

이름붙는 반복문
-> 반복문에 이름을 붙여서 
     하나 이상의 반봇문을 벗어날 수 있다.
-> break; 
     == 하나의 반복문을 벗어난다.!!
